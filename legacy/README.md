# legacy/

الملفات دي كانت موجودة في المشروع القديم بس مش مستخدمة فعليًا في التطبيق الحالي
(الـ app.js الجديد ولا أي route ما بيعملهاش require):

- **agentsRepo.js** — نظام موظفين قديم (`NileChat_Agents_byA`) اتبدل بالكامل بنظام
  `NileChat_Users_byA` الموجود دلوقتي في `src/repositories/user.repo.js`. لسه مستخدم
  بس من `scripts/seedAgent.js` (سكريبت قديم منفصل عن السيرفر).
- **helpers.js** — دوال قديمة كانت بتستخدم Postgres (`pool.query(...)` بصيغة `$1, $2`)
  رغم إن المشروع دلوقتي شغال بالكامل على SQL Server (`mssql`). غير متوافقة مع باقي
  الكود ومحدش بيعملها require.
- **whatsapp.js** — نسخة قديمة وأبسط من إرسال رسائل واتساب، اتبدلت بـ
  `src/services/whatsapp.service.js` (اللي بيدعم Inboxes متعددة وبيسجل الرسالة في الداتابيز).

سيبتهم هنا بدل ما أمسحهم نهائيًا، تقدر تراجعهم أو تمسحهم براحتك لو مش محتاجهم.
